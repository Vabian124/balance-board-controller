## Reporting a vulnerability

Please **do not** open a public GitHub issue for security-sensitive bugs.

1. Open a [private security advisory](https://github.com/Vabian124/balance-board-controller/security/advisories/new) on GitHub, or
2. Email the repository owner via their GitHub profile contact.

Include steps to reproduce, affected versions, and impact if known. You should receive a response within a reasonable timeframe.

This app runs locally on Windows, uses Bluetooth/HID, and can emit virtual joystick and keyboard input — treat unexpected remote control or privilege escalation as high severity.
